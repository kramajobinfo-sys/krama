# Public website — UI kit

Interactive recreation of Krama's public-facing site. Single-page click-through; navigation is in-memory React state.

## Screens
- **Home** (`home.jsx`) — hero with dual search, job categories grid, featured jobs, featured companies, employer CTA band.
- **Jobs search** (`jobs.jsx`) — sticky search strip, filter sidebar (category / type / work mode), sortable results list, applied-filter tokens.
- **Job detail** (`job-detail.jsx`) — full posting, key-facts row, requirements/benefits, sticky apply card, company card, similar jobs.

## Composition
Built entirely from design-system primitives via `window.KramaDesignSystem_1a6f65`: `JobCard`, `CompanyCard`, `Card`, `Button`, `Badge`, `Tag`, `Avatar`, `Select`, `Checkbox`. Shared chrome (`chrome.jsx`) provides the header + footer. Sample content lives in `data.js`. Icons via Lucide CDN; brand marks + weave from `assets/`.

## Run
Open `index.html`. Requires the compiled `_ds_bundle.js` at project root (generated by the design-system compiler).
