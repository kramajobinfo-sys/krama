The primary listing unit — use in search results, featured grids, "similar jobs", and saved-jobs lists. Set `featured` for promoted posts.

```jsx
<JobCard title="Senior Accountant" company="ABA Bank" location="Phnom Penh"
  salary="$800 – 1,200 / mo" type="Full-time" remote postedAt="2h ago"
  featured saved onSave={toggle} onClick={open} />
```
